# App Description:
<br>
Mutli-Showdown is a web-based showdown game that has characters from Pok√©mon, DnD, and  YuGiOh universes go up against one another in statistically fair combat. You are able to create your own deck of characters to fight bots. The site normalizes each character's stats for turn-based battles with type advantages, and displays everything dynamically.
